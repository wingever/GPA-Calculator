# GPA-Calculator
基于react、webpack、bootstrap完成的一个GAP计算器，gpa计算部分比较随便，主要是项目实现，引入包等

## 项目运行
+ 下载项目

  > git clone https://github.com/wingever/GPA-Calculator.git
  
+ 安装依赖

  > npm install
+ 运行项目
  
  > npm start
